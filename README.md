# Razorpay Landing Page Clone

A UI clone of the Razorpay landing page built to demonstrate modern web design principles, responsiveness, and utility-first CSS.




## ✨ Features
- **Fully Responsive:** Adapts seamlessly to Desktop, Tablet, and Mobile devices using Tailwind's breakpoint system.
- **Modern UI:** High-fidelity design mirroring the original Razorpay website.
- **Custom Animations:** Smooth, infinite keyframe animations (e.g., the vertical scrolling company logos).
- **Interactive Elements:** Engaging hover states and transitions on buttons and feature cards.

## 🛠️ Tech Stack
- **HTML5** for semantic page structure
- **Tailwind CSS** (via CDN) for rapid, utility-first styling
- **FontAwesome** for scalable vector icons

## 🏃‍♂️ Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/razorpay-clone.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd razorpay-clone
   ```

3. **Open the project:**
   Simply double-click the `index.html` file to open it in your web browser, or use an extension like **Live Server** in VS Code for hot-reloading.
